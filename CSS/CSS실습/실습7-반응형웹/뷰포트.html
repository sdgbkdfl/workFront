em과 rem

font-size에 비례해서 결정되는 상대 단위
em : 요소의 위치를 기준으로 1em
rem : 최상위 요소(html)의 글자 크기를 기준으로 1rem

상대단위 : 고정되지 않고 어떤 기준에 따라 유동적으로 바뀔 수 있는 길이(em,rem,%,vw,vh)
고정단위 : 항상 고정된 길이를 나타내는 단위(px,pt)
