1. 모든 선택자(*)

HTMl 문서안의 모든 태그를 선택할 때 사용 여러개 스타일 지정 가능
(pre태그 : 작성한 그대로 화면 구현) 
        *{
            스타일 속성:값;
            ...
        }

2. 태그 선택자

태그명을 이용해 요소 선택 
        태그명{
            스타일 속성:값;
        }

3.아이디 선택자

요소의 id속성을 이용하여 요소 선택 #아이디 속성 값 
        #아이디명{
            스타일 속성:값;
        }

4.클래스 선택자(.class명)

HTML문서 내에 여러개의 태그를 동일한 클래스 명으로 지정하여 선택할 때 사용 
        .클래스명{
            스타일 속성:값;
        }
