1 자료형

자바스크립트에서는 변수에 별도로 자료형을 지정해주지않고 리터럴에 의해 결정
리터럴 : 변수에 대입되는 값 그 자체

자료형을 확인하는 방법
typeof(값 또는 변수명)

Primitive type 원시타입

하나의 값만 저장하는 자료형
  • string(문자열)

    • '',""로 묶은 데이터

  • number(숫자)

    • 정수와 실수 모두 포함

    숫자가 들어있어도 따옴표로 묶이면 문자열로 인식

  • boolean (논리값)

    • true, false

    falsy : false로 인정할 수 있는 값
    boolean(0) : boolean타입으로 형변환
    0, '',"" `` : 빈문자열 NaN, undefined, null
    res=0?'true':'false'

    truthy : true로 인정 할 수 있는값 (falsy를 제외한 값)
  • undefined

    • 변수를 선언만하고 값을 할당하지않았을 때 변수의 초기값

  • null

    • 값이 없거나 유효하지 않은 값
